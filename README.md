# 功能说明

###  1. 如何连接
在iOS越狱手机安装“flybirdremotecontrol_1.0.deb”，pc电脑下载“飞鸟群控PC端.zip”解压缩，打开flybird.exe,通过手机端的“飞鸟群控”连接。手机和PC必须处于同一局域网内。

###  2.主要功能
通过PC端控制多台iOS越狱手机，支持控制手机50-100台
PC端可以实时控制手机，功能请查看PC端截图。

支持鼠标点击、滑动操作。
